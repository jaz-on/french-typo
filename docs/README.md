# French Typo Documentation

This folder contains the complete documentation for the French Typo plugin, intended for developers and AI tools like Cursor.

> **Note**: For user documentation intended for WordPress.org, see the [`readme.txt`](../readme.txt) file at the root of the project.

## Documentation Organization

### Developer Documentation

- **[Architecture](architecture.md)** — Detailed technical architecture of the plugin, code structure, processing flow, design decisions
- **[API](api.md)** — Complete public API documentation, hooks, filters, and usage examples
- **[Development](development.md)** — Local development guide, code conventions, Git workflow, testing and validation

### User Guides

- **[Installation](installation.md)** — Detailed installation guide, installation methods and prerequisites
- **[Configuration](configuration.md)** — Complete configuration and settings guide for the plugin
- **[FAQ](faq.md)** — Frequently asked questions and detailed answers

## About This Documentation

This documentation is maintained in the project's Git repository and is versioned with the code. It is automatically accessible to AI tools like Cursor to provide context during code generation.

## Contributing

To contribute to the documentation:

1. Edit the Markdown files in this `docs/` folder
2. Follow French typography rules (non-breaking spaces, etc.)
3. Reference code files with their line numbers when relevant
4. Maintain consistency with the source code
